# App Store screenshots

Capture on-device (or Simulator + framing). Do **not** commit binary screenshots to git unless deliberately approved (large binaries).

## Required slots (iPhone)

| Display | Portrait sizes (px) |
|---------|---------------------|
| 6.7" (iPhone 15 Pro Max class) | 1290 × 2796 |
| 6.1" (iPhone 15 / 14 class) | 1179 × 2556 |

Optional iPad if you enable iPad in ASC.

## Shot list

1. Hub chat (home) — brand + channel visible  
2. Discover / join hubs  
3. DM thread  
4. Voice dock connected  
5. Friends  
6. Settings → Account (export / delete visible)

## Rules

- Dark Nexus UI; EN (add AR locale set separately)  
- No “mock mode”, no competitor logos, no watermark  
- Use real TestFlight build against production URL  

## Local folder (gitignored suggestion)

Put exports in `docs/store/screenshots/en-US/` (create locally; keep out of remote if huge).
